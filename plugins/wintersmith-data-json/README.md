wintersmith-plugin
==================

[Wintersmith](https://github.com/jnordberg/wintersmith) plugin template

## Installing

Install globally or locally using npm

```
npm install [-g] wintersmith-plugin
```

and add `wintersmith-plugin` to your config.json

```json
{
  "plugins": [
    "wintersmith-plugin"
  ]
}
```

## Running tests

```
npm install
npm test
```
